## Flutter-Firebase-Projects

This repository contains a collection of mobile application labs and practice projects developed using Flutter, Dart, and Firebase. Each project focuses on core mobile development concepts such as UI design, application logic, and backend integration.

---

### 1. Dice Rolling App
A turn-based dice game where players attempt to reach a target score within a limited number of rolls.

**Key Features**
* Visual dice output with placeholder state before rolling
* Remaining roll counter updated in real time
* Live score accumulation display
* Progress indicator showing distance from target
* End-of-round feedback with restart option

**Skills Practiced**
* Managing widget state and rebuilds
* UI updates driven by game logic
* Conditional rendering based on app state
* Basic game flow control
